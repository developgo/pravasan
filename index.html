PRAVASAN

Simple Migration Tool.


Intend to be used for any programming languages, for any database.
Its still in beta. Testing in progress.*
Written in Go.

Try it


  * Currently only available for MySQL. Check Here if your database is in our pipeline.

Usage

Below are the simple commands which will help you to create the migration file in JSON format.

$ ./pravasan add create_table test123 id:int name:string order:int status:bool    # Create JSON Migration File for TABLE Creation
$ ./pravasan add add_column test123 id:int     # Create JSON Migration file containing Add Column syntax
$ ./pravasan add drop_column test123 id        # Create JSON Migration file having Drop Column syntax 
$ ./pravasan create                            # Create `schema_migrations` table

$ ./pravasan up                                # Execute migration files which is not been executed.

$ ./pravasan down [-1]                         # Revert Migration file which is been executed already.
